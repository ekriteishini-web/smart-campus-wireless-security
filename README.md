Smart Campus Wireless Security Architecture
Overview

This project demonstrates the design, implementation, and security evaluation of a Smart Campus Wireless Network using Cisco Packet Tracer and Wireshark.

The objective was to create a secure WLAN environment supporting students, staff, and IoT devices while evaluating common wireless attacks and implementing appropriate mitigation techniques.

Key Features
- Secure WLAN architecture design
- Wireless network segmentation
- IoT device integration
- Attack simulation and analysis
- Security control evaluation
- Traffic monitoring using Wireshark
  
Technologies Used
- Cisco Packet Tracer
- Wireshark
- Wireless Networking Concepts
- WLAN Security Controls
  
Project Deliverables
- Network topology and configuration (.pkt)
- Technical report (.pdf)
- Attack simulation demonstration (.mp4)
  
Skills Demonstrated
- Wireless Network Security
- Network Design and Architecture
- Traffic Analysis
- Security Monitoring
- Attack Detection and Mitigation
- Technical Documentation
